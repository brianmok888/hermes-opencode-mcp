# Legacy bridge archive

This directory is a preservation archive of the former `hermes-opencode-bridge` repository.

It is kept inside `hermes-opencode-mcp` so the MCP repository contains all implementation and historical operational assets before the old bridge repo is deleted.

## Status

- canonical implementation: `hermes-opencode-mcp`
- archived source of removed repo: `legacy/hermes-opencode-bridge/`
- old standalone bridge repo: intended for deletion after this archive is committed and pushed

## What is preserved here

- original bridge source tree
- original bridge tests
- original templates
- original top-level docs and packaging files

This archive is for reference, migration traceability, and cleanup safety. New feature work should not be added here.
